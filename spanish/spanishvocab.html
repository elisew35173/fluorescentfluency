Spanish Vocabulary

Click on the image to go the resource.


Duolingo

Resource Type: App and Website

Teaches you basics of vocabulary by dividing learning by topics, such as colors, food, sports, and family members.

Merriam-Webster's Spanish-English Dictionary

Resource Type: Website

It is always a good idea to have a dictionary on hand when learning a language. This allows you to look up words anytime anywhere without relying on an internet connection or a nearby translator.

SpanishDict

Resource Type: Website

A simple and easy to use website to help you translate words. A tool such as this is good to use if you ever forget a vocabulary word or need to look up a vocabulary word you saw somewhere!

Quizlet

Resource Type: App and Website

Find Spanish vocabulary sets created by other users to learn through practice exercises and games. Or, create your own vocabulary sets of Spanish words that you want to master.

SpanishPod101

Resource Type: App and Website

This website allows you to listen to dialogue and then cover the grammar points and vocabulary. It is more on the advanced learner side.

Conjuguemos

Resource Type: Website

A website created by language teachers, they have vocabulary lists sorted by topic and by textbook. Each list comes with graded practice and activities such as crosswords. The site is also good for grammar and listening.
